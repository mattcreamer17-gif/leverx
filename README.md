# LeverX landing page

A single-page methodology brief prepared for the LeverX AI practice. The tone is peer-to-peer: one consulting firm walking another through a scoping framework for Joule, BTP, and the layers underneath. Designed to look native to LeverX's own site (dark navy header, Roboto Condensed headings, Open Sans body, amber CTA) so the booth moment reads as "your marketing team built this."

Intended URL path: /for/leverx

## Files

- `index.html` — the page
- `styles.css` — tokens drawn from `brand-pack.json`
- `logo.svg` — LeverX mark, copied from the brand pack
